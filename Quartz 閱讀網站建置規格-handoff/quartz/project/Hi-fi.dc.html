OpenCode MCP 配置指南

2026-06-07· #tool #mcp #opencode
本筆記記錄 OpenCode 已配置的 MCP servers，設定檔位於 ~/.config/opencode/opencode.json。

已配置的 MCP Servers

Server用途狀態
obsidian讀寫 Obsidian vault已啟用
notebooklmNotebookLM 筆記本管理已啟用
playwright瀏覽器自動化已啟用
open-computer-use桌面自動化（點擊、輸入、截圖）已啟用

opencode.json 完整 MCP 設定

opencode.json複製
{
  "mcp": {
    "obsidian": {
      "type": "local",
      "command": ["mcpvault.cmd", "D:\\my-note"],
      "enabled": true
    },
    "playwright": {
      "type": "local",
      "command": ["npx", "-y", "@playwright/mcp"],
      "enabled": true
    }
  }
}

各 Server 安裝說明

NotebookLM

  • 套件：notebooklm-mcp-cli v0.7.1
  • 安裝：uv tool install notebooklm-mcp-cli
  • 重新登入：nlm login（需關閉 Chrome 後執行）
  • 文件：oc-go-cc 設定指南
💡 登入小技巧
若 nlm login 瀏覽器閃退，先執行 Stop-Process -Name chrome -Force 關閉所有 Chrome 後再試。

重新啟動說明

OpenCode 配置只在啟動時載入一次，修改設定後需重啟才生效。重啟後右上角會出現對應數量的 MCP 燈號。
